# Weather Skill - 天气查询技能

## 描述

查询指定城市的天气信息，包括温度、天气状况等。

## 何时使用

- 用户询问天气情况
- 用户计划出行需要天气信息
- 用户提到"天气"、"温度"、"下雨"等关键词

## 工具

### get_weather

获取指定城市的天气信息。

**参数**：
- `city` (string, 必须): 城市名称，如"上海"、"北京"
- `days` (number, 可选): 预报天数（1-7），默认 1

**示例**：
```json
{ "city": "上海", "days": 3 }
```

**返回**：
```json
{
  "city": "上海",
  "current": {
    "temp": 25,
    "condition": "晴",
    "humidity": "60%"
  },
  "forecast": [
    { "date": "2024-01-01", "temp": "23-28", "condition": "多云" }
  ]
}
```
