# Search Skill - 搜索技能

## 描述

搜索互联网获取实时信息，适用于查询新闻、百科知识、技术文档等。

## 何时使用

- 用户询问实时信息（新闻、股价等）
- 用户询问特定事实或数据
- 用户明确说"搜索"、"查一下"
- 用户的询问超出你的知识范围

## 工具

### web_search

搜索互联网内容。

**参数**：
- `query` (string, 必须): 搜索关键词
- `maxResults` (number, 可选): 最大结果数（1-10），默认 5

**示例**：
```json
{ "query": "2024年奥运会举办城市", "maxResults": 3 }
```

## 注意事项

1. 搜索关键词应该简洁明了
2. 结果可能需要筛选和整理
3. 对于时效性强的信息，优先使用搜索
