# Chat Expert Instance - 专家副本系统

## 概述

这是一个单专家副本系统，每个副本代表一个独立的 AI 专家。通过将副本部署到不同目录或机器，多个专家可以通过消息平台进行对话。

**核心理念**：
- 每个副本只代表一个专家
- 所有副本共享相同的数据库配置
- 通过数据库配置定义专家属性
- 通过消息平台与其他专家通信
