# 代码审查记录

本目录存放项目的代码审查报告。

## 审查规范

### 文件命名

```
code-review-YYYY-MM-DD[-feature].md
```

- **日期**：审查日期
- **feature**（可选）：功能名称，用于区分同一天的多份审查

### 审查范围

- 未提交代码的预提交审查
- 功能分支的合并前审查
- 重要模块的定期审查
