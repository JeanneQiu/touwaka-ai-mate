# 知识库界面优化

## 需求描述

优化知识库详情页面的用户体验：

1. **移除顶部"新建文章"按钮** - 简化界面，新建文章功能可通过树节点右键菜单实现
2. **移除右键菜单** - 减少用户困惑，操作按钮更直观
3. **重新设计标题行布局** - 编辑和删除按钮放在标题旁边，排版更清晰
4. **删除按钮防误触设计** - 默认低透明度显示，hover 时才高亮

## 实现方案

### 界面变更

- 移除 `detail-header` 中的"新建文章"按钮
- 移除 `KnowledgeTreeNode.vue` 中的右键菜单功能
- 在文章标题行添加编辑和删除按钮
- 删除按钮使用较低透明度（opacity: 0.4），hover 时恢复正常

### 文件修改

- `frontend/src/components/KnowledgeTreeNode.vue` - 移除右键菜单
- `frontend/src/views/KnowledgeDetailView.vue` - 重新设计标题布局

## 删除按钮防误触设计

1. **低透明度**：默认 opacity: 0.4，不显眼
2. **距离编辑按钮有一定间隔**：8px gap
3. **Hover 效果**：鼠标悬停时背景变浅红色，提醒用户这是删除操作
4. **确认对话框**：点击后仍需确认才能删除
